# Page Fetcher

## Run 
- In your terminal type 

```javascript
1 - node 
2 - fetcher.js
3 - http://www.example.edu/
4 - ./index.html
```
- Run all together
```javascript 
node fetcher.js http://www.example.edu/ ./index.html 
```
- Check the new file created in your directory
```javascript 
./index.html 
```
